Turn multilingual PDFs into readable TXT.

BhashaLens OCR converts PDFs and DOCX files into clean UTF-8 text files using a hybrid pipeline: fast embedded-text extraction first, then Tesseract.js OCR when a page is scanned, empty, or likely garbled. Output files keep the original base filename and are bundled into one ZIP for bulk work.

Methodology

  1. PDF.js text layer extraction reads embedded text first and reconstructs lines by page coordinates.
  2. Legibility scoring flags empty pages, replacement symbols, private-use glyphs, and common Indic mojibake.
  3. Tesseract.js OCR fallback renders flagged pages to canvas and recognizes selected languages with a browser worker.
  4. UTF-8 TXT export normalizes spacing and saves each result with the same base filename.

Privacy model

All document processing is client-side. Your files and extracted text are not exchanged with a server by this app.

For first-time OCR use, the browser may download open-source JavaScript, WebAssembly, and Tesseract traineddata from public CDNs. To run in a fully offline environment, self-host those dependencies and update the paths in the HTML.

Best results

  • Select only the languages actually present in the file.
  • Use Smart hybrid for mixed digital/scanned PDFs.
  • Use Deep OCR for scans, bad fonts, or unreadable embedded text.
  • Increase render scale for old, low-resolution, or photocopied scans.
